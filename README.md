# Terminal Messenger v3.0

🔐 Анонимный мессенджер в стиле терминала с шифрованием и полным набором команд.

## 🎯 Возможности

- **Аккаунты** — регистрация, вход, смена пароля
- **Посты** — создание, чтение, удаление, экспорт
- **Чат** — отправка и чтение сообщений (с шифрованием)
- **Команды** — `ping`, `fortune`, `tree`, `theme`, `game`, `encrypt`, `decrypt` и др.
- **Друзья** — добавление, список
- **Статистика** — количество постов, сообщений, игр
- **Темы** — green, amber, white, matrix
- **История команд** — стрелочки вверх/вниз

## 🚀 Запуск

Просто откройте `index.html` в браузере или зайдите на [GitHub Pages](https://ваш-ник.github.io/terminal-messenger/).

## 🛠 Технологии

- HTML / CSS / JavaScript
- LocalStorage для хранения данных
- Простое шифрование XOR + Base64

## 📝 Команды
help — список команд
login <login> <pass> — войти
register <login> <pass> — создать аккаунт
logout — выйти
whoami — информация о себе
users — список пользователей
ls — список постов
post <title> — создать пост
img <title> — пост с картинкой
read <id> — прочитать пост
rm <id> — удалить пост
chat — показать чат
say <text> — отправить сообщение
ping <host> — проверить связь
fortune — случайная цитата
tree — дерево постов
passwd — сменить пароль
theme <name> — сменить тему
export — экспорт постов
friend add <login> — добавить в друзья
friend list — список друзей
stats — статистика
game guess — угадай число
encrypt <text> — зашифровать
decrypt <text> — расшифровать
clear — очистить экран
